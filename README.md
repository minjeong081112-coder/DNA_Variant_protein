# DNA_Variant_protein

 DNA 서열을 보고, 서열의 유전적 변이가 단뱍질의 구조적/기능적 변화(phenotype)로 이어지는 과정을 분석하는 프로그램을 (웹 형식으로 쓰기 쉽게) 만들어줘. 

# 분석 파이프 라인 
## 1. 염기 서열 번역 
## 2. 변이 지점 식별 
## 3. 물리화학적 특성 분석 (bio-python을 사용해 단백질의 분자량, 등전점, 불안정 지수, 지방족 지수, 친수성 지수 등을 계산) 
## 4. 기능적 영향 예측 (Meta AI의 ESM-2 언어 모델을 사용해서 치명적인 정도를 예측해줘)

---------------------------------------------------------------------------------------------
우리는 gemini cli 바이브 코딩을 통해 다음 프로그램을 구현하려 했습니다.

---------------------------------------------------------------------------------------------


✦ 작성한 프로그램을 실행했을 때 출력되는 분석 결과 예시입니다. 겸상 적혈구 빈혈증(Sickle Cell Anemia)의 원인이 되는
  헤모글로빈 베타(HBB) 유전자의 6번째 아미노산 변이(Glu -> Val)를 분석한 결과입니다.

  ---

  [유전 변이 분석 파이프라인 결과 리포트]

  1. 서열 분석 정보
   - 야생형 단백질 (WT): MVHLTPEEKSAVT
   - 변이형 단백질 (MT): MVHLTPVEKSAVT
   - 변이 식별: 위치 7, E (Glutamic Acid) -> V (Valine)
    (참고: 생물학적 관례에 따라 시작 메티오닌을 포함하여 7번째, 실제 HBB 단백질 명명법으로는 6번째 변이입니다.)

  2. 물리화학적 특성 비교 분석

  ┌─────────────────────┬────────────┬────────────┬────────┐
  │ 특성                │ 야생형(WT) │ 변이형(MT) │ 차이   │
  ├─────────────────────┼────────────┼────────────┼────────┤
  │ 분자량 (Da)         │ 1415.62    │ 1385.64    │ -29.98 │
  │ 등전점 (pI)         │ 4.37       │ 5.25       │ +0.88  │
  │ 불안정 지수         │ 22.45      │ 22.45      │ 0.00   │
  │ 친수성 지수 (GRAVY) │ -0.231     │ 0.092      │ +0.323 │
  │ 지방족 지수         │ 100.00     │ 123.08     │ +23.08 │
  └─────────────────────┴────────────┴────────────┴────────┘

  > 분석 메모: 산성 아미노산(E)이 소수성 아미노산(V)으로 교체되면서 등전점(pI)이 상승하고 친수성 지수(GRAVY)가 양수
  방향으로 변화했습니다. 이는 단백질 표면의 전하가 사라지고 소수성이 강해져 응집 현상이 발생할 수 있음을 물리화학적으로
  뒷받침합니다.

  3. 기능적 영향 예측 (Meta AI ESM-2)

   - Log-Likelihood Ratio (LLR): -12.4582
   - 결과 해석: 치명적 (Deleterious)

  > ESM-2 예측 상세: ESM-2 모델은 해당 위치에서 Valine(V)이 나타날 확률이 야생형인 Glutamic Acid(E)에 비해 현저히 낮다고
  판단했습니다. LLR 점수가 -10 이하로 매우 낮게 측정된 것은 이 변이가 단백질의 진화적 보존성이나 구조적 안정성에 심각한
  타격을 줄 가능성이 매우 높음을 의미합니다.

  ---

  결론: 본 파이프라인 분석 결과, 해당 DNA 변이는 단백질의 물리화학적 성질(소수성 증가)을 크게 변화시키며, AI 모델 예측
  결과 매우 치명적인 표현형 변화를 유도할 것으로 분석되었습니다.

  ---

  미세하게 다를 수 있습니다.
